基本概念與架構

為什麼使用C++

C++相較其他程式語言來說，是相對容易學習的，所以適合當作第一個學習的程式語言！

  • 它是一種物件導向的程式語言，可以為程式提供清晰的結構，並允許程式碼被重複使用，降低開發成本。

  • 可以用於開發可以適應多個平台的應用程式。

  • 可以在現今的作業系統、圖形使用者介面以及嵌入式系統中找到。

Hello World !

#include<iostream>
using namespace std;
int main()

{
    cout << "Hello,World!" << endl;
    return 0;
}
  • 第一句 #include<iostream>
    • <>中的檔案，中文被稱做標頭檔(header file)，這些標頭檔可能是已經定義的函數或是變數，或是一些通用的東西(library)。例如是在C++中的standard library 中已經定義好的input & output。 #include<iostream>本身就是匯入這些定義好的東西，讓compiler可以正常運作。

  • using namespace std;
    • 這句話的重點在namespace這個詞，中文被翻譯成命名空間。 這個東西是作為儲存名稱用的，當今天程式要寫出像是cout 或是cin等的程式碼時，如果沒有使用using namespace std; 就會讓程式無法成功運作。而其中std就是一個叫做standard的命名空間。
  • 主函數(main function)，int main(){} 代表的是一種會回傳整數給電腦的函數。

    循序執行： 將程式碼放入{ }中，就會由上到下執行。

  • 結束：一旦執行到return 0;，程式就會立刻結束。

    • 這一行的存在對於主函數而言很重要！它的作用是讓函數的進行終止，後面的0也可以改成其他的數值(即改變回傳的值)，但是有些主函數(void main())是不需要回傳值的，但因為我也還沒接觸到所以先把他跳過。在程式讀到return 0;的時候，就會像是返回了一個標誌，表示程式成功的執行完成了，可以退出了

  • 排版：在{ }內的東西，可以利用 Tab鍵 向右推。
