二維陣列 Array

二維陣列使用陣列名稱與兩個索引值來指定存取陣列元素，宣告方式與一維陣列類似： 

資料型態 陣列名稱[陣列長度][陣列長度];

實際程式碼 :

int maze[5][10];

上面這個宣告會配置 5 * 10 = 50 個整數的記憶體空間給陣列來使用，二維陣列使用兩個索引值來指定存取陣列， 這兩個索引值都是由 0 開始，下面這個程式簡單的示範二維陣列的存取：

#include<iostream>
using namespace std;    
int main(){

    constespr int rows[5];
    constespr int columns[10];
    int maze[rows][columns];
    
    for (int row=0;row < rows; row++){
        for(int i =0;i < columns; i++){
            maze[row][i] = (row + 1) * (i + 1);
        }
    }

    for (int row=0;row < rows; row++){
        for(int i =0;i < columns; i++){
            cout << i << " : " << maze[row][i] << endl;
        }
        cout << endl;
    }
    
    return 0;
}

output

0 : 100
1 : 500
2 : 600
3 : 200

範例 - 依序輸入陣列、依序輸出陣列

  • 陣列長度直接宣告成題目敘述的最大範圍就好，本題的陣列長度至少為1000

  • 輸入要用到一次while，輸出也要用到一次while

輸入說明

第1列：整數N，代表接下來要輸入N個數字。 ( N <= 1000 )

第2列：N個整數。

輸出說明

依序輸出被輸入的N個整數，以空白隔開。

input

12
2200 1500 300 -800 600 1750 8220 6720 0 -450 720 3590

output

2200 1500 300 -800 600 1750 8220 6720 0 -450 720 3590

code

#include<iostream>
using namespace std;

int main(){
    int N;
    int num[1005];
    int i;

    // input
    cin >> N;

    i = 0;
    while( i < N )
    {
        cin >> num[i];
        i = i+1;
    }

    // output
    i = 0;
    while( i < N )
    {
        cout << num[i] << " ";
        i = i+1;
    }
    cout << endl;

    return 0;
}
