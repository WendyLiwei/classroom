C++中不同的變數

  • int : 整數，舉例：1、0、123 ...

  • float : 浮點數，舉例：3.14159、0.6...

  • char : 字元，舉例：'A'、'X'、'7'...

  • string : 字串，舉例："Hello"、"Canada" ...

  • bool : 布林值，舉例：true、false ...

cin >> 和 cout <<

#include<iostream>
using namespace std;
int main()
{

    int number = 0 ;
    cin  >> number = 0 ;
    cout << number ;

    return 0;
}
  • cin >> 表示輸入
  • cout <<表示輸出
    • 當輸入100時，按下Enter，螢幕就會顯示 100 了
  • cin >> 裡面，原本宣告的變數是什麼型態的，輸入的東西就要是相同型態。
